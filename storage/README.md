# storage/

Runtime file storage for PolyDoc AI (Phase 1 local development).

| Directory | Purpose |
|-----------|---------|
| `uploads/` | Original uploaded PDFs and images |
| `documents/` | Serialized JSON document models |
| `outputs/` | Exported PDF, DOCX, and HTML files |
| `models/` | Cached ML model weights (PaddleOCR, DocLayout-YOLO) |

Contents are gitignored except `.gitkeep` placeholders.
